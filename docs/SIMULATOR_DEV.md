# Запуск в симуляторе с автообновлением

## Один раз: установить приложение

```bash
# Удалить старое приложение с симулятора (опционально)
xcrun simctl uninstall booted com.anonymous.ai-lawyer

# Собрать и установить (Metro запустится с поддержкой live reload)
npm run ios:dev
```

Дождись окончания сборки и появления приложения в симуляторе.

## Чтобы правки в коде сразу были видны в симуляторе

1. **Оставь терминал с `npm run ios:dev` открытым** — в нём работает Metro. Пока он запущен, приложение в симуляторе подгружает JS с твоего компьютера.

2. **Сохраняй файлы** — через 1–2 секунды приложение в симуляторе само обновится (Fast Refresh).

3. Если обновление не сработало — в симуляторе нажми **⌘R** (Cmd+R) для ручной перезагрузки.

## Если запускаешь Metro отдельно

В первом терминале:

```bash
npm run start:dev
```

Во втором (когда Metro уже «Listening»):

```bash
npx expo run:ios --no-bundler
```

Приложение установится и будет подключаться к Metro из первого терминала — правки в коде будут сразу видны.
