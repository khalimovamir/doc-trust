# Мгновенное обновление в Simulator (Fast Refresh)

Чтобы при сохранении файлов изменения сразу показывались в симуляторе:

## Шаг 1. Запустить Metro с очисткой кэша

```bash
npm run start:clear
```

Оставь этот терминал открытым.

## Шаг 2. Запустить приложение в симуляторе

Во **втором терминале** (один раз):

```bash
npm run ios:simulator
```

## Шаг 3. Подключить приложение к Metro

В симуляторе откроется экран «Doc Trust Development Build». **Нажми на строку `http://localhost:8081`** — приложение подгрузится с Metro.

## Шаг 4. Включить Fast Refresh

В симуляторе нажми **Cmd + D** → выбери **Enable Fast Refresh**.

После этого при **сохранении любого файла** (JS/TS/TSX) экран в симуляторе обновится сам, без перезапуска.

## Если обновления не появляются

- Убедись, что Metro запущен первым (`npm run start:clear`), затем запущен симулятор.
- Убедись, что в приложении нажали на **localhost:8081** (приложение грузится с Metro).
- Проверь, что в меню (Cmd+D) включён **Enable Fast Refresh**.
- Попробуй перезапустить Metro с `npm run start:clear` и в симуляторе нажать **Cmd+R** (Reload).
